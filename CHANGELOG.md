# BSICARDS Android SDK - Changelog

All notable changes to the BSICARDS Android SDK will be documented in this file.

## [1.0.0] - 2026-03-06

### Added
- Initial release
- Complete MasterCard operations (create, get, freeze, fund, etc.)
- Complete Visa Card operations (create, get, freeze, fund, etc.)
- Complete Digital Wallet operations (create, get, fund, USDT address, etc.)
- Utility operations (wallet balance, deposits, transactions)
- Kotlin Coroutines for async operations
- Retrofit 2 integration for REST API
- Comprehensive error handling with custom exceptions
- Complete documentation and examples
- Unit tests
- ProGuard rules for production builds

### Features
- Android 5.0+ support
- Kotlin first approach
- Type-safe API with data classes
- Secure credential management
- Input validation
- Network error handling
- Suspend functions for coroutines
- Gson JSON serialization
